How to Export Your Data

Life Coach AI by Spry

Steps to Export

  1. 1 Open Life Coach AI on your device
  2. 2 Go to the Profile tab (last tab in the bottom navigation)
  3. 3 Scroll down to the Data Management section
  4. 4 Tap "Export Data" (blue button)
  5. 5 The app will generate a JSON file and open the device share sheet
  6. 6 Choose where to save or send the file — you can save it to Files, email it to yourself, upload to cloud storage, etc.
File format: Your data is exported as a single JSON file (life-coach-export.json) that can be opened with any text editor or JSON viewer.

What Is Included

The exported file contains all of your locally stored data, organized by category:

Category What It Contains
Sessions Coaching session records including start/end times and summaries
Messages All chat messages between you and the AI coach, organized by session
Tasks CBT tasks assigned by the coach, including type, status, and completion dates
Journal Entries Your written journal reflections linked to specific tasks
Tracking Entries Numerical tracking data (e.g., mood scores, habit counts) linked to tasks

What Is Not Included

The following data is not part of the export:

  • Account credentials — Your email and password are managed by Firebase and are not stored locally
  • Profile data — Name, age, gender, focus areas, and personality scores are stored in Firebase Firestore (contact us if you need a copy)
  • Subscription status — Managed by your app store and RevenueCat
  • Daily mood check-ins — Stored locally but not included in the current export (contact us if needed)
  • PHQ-9 assessment results — Stored locally but not included in the current export (contact us if needed)
